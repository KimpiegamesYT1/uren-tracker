# Uren Tracker

Een simpele Expo app om uren en onkosten bij te houden.

## Wat je met de app kunt

- Uren registreren per bedrijf
- Onkosten toevoegen
- Maandoverzichten bekijken
- Openstaand saldo en betalingen bijhouden
- Donker en licht thema gebruiken

## Techniek

- Expo + React Native
- Expo Router
- TypeScript
- Lokale SQLite database
